# Calculadora Simples

## Crei esse projeto para me desafiar e colocar em prática o uso de JavaScript em conjunto com HTML e CSS
O projeto é muito simples e consiste em fazer uma calculadora com histórico de operações em uma página web(eu sei que é bem melhor e mais acessível usar uma calculadora de celular ou do próprio pc, mas como eu disse, esse projeto foi para me testar).
Me inspirei na calculadora do Windows11 para deixar fluído e lógica de operações sequênciais, e o desing eu me inspirei de um projeto que encontrei no figma
